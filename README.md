CRM Lead Management System

Overview

This is a CRM (Customer Relationship Management) web application developed using React.js, Node.js, Express.js, and MongoDB.

The system helps manage customer leads, track follow-up dates, update lead status, and maintain notes.

Features

- User Login Authentication
- Logout Functionality
- Add New Leads
- View All Leads
- Update Lead Status
- Delete Leads
- Follow-up Date Management
- Notes Management
- Responsive UI

Tech Stack

Frontend

- React.js
- CSS

Backend

- Node.js
- Express.js

Database

- MongoDB Atlas
